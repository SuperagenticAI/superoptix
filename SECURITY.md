## Reporting a Vulnerability

We take the security of SuperOptiX very seriously. If you discover a security vulnerability, please follow these steps:

1.  **Do not open a public issue.** This gives malicious actors a chance to exploit the vulnerability before we can fix it.
2.  **Email us directly.** Please send a detailed report to [security@super-agentic.ai](mailto:security@super-agentic.ai).
3.  **Include details.** Please provide as much information as possible, including:
    *   Steps to reproduce the vulnerability.
    *   Versions of SuperOptiX and Python being used.
    *   Any relevant configuration files or code snippets.

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix. We appreciate your cooperation in disclosing vulnerabilities responsibly.
